# Examples

## Setup

**Install dependencies**

```
cd examples
cp template.env .env
npm install
```

Modify .env and add your private key

Rest of this README assumes you are in the `examples` directory

## Running

**Chat GPT example**
```
npm run chat
```

**Chat Vision example**
```
npm run chat_vision
```

**Anthropic Claude example**
```
npm run chat_anthropic
```

**Agent example**
```
npm run agent
```
